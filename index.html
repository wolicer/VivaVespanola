Memo

Introduction:

Pioneer's Daily Memo is a page designed to help Granado Espada players in keeping track of daily online activities.

Notes:

  • This page uses browser local storage to store data. Consider using a different browser if your browser doesn't support local storage. This also means data is stored locally and will not be transferred across the web or across browsers.
  • This page mainly uses client side programming which uses a lot of scripts. It loads slower but you never have to reload/refresh the page once it is loaded. Use the Refresh/Sort button instead to update the record's sort and order.
  • Use the Refresh/Sort button to quickly update the corresponding tables which also sorts it by predetermined order. Adding, editing or deleting any record also does the refreshes and sort the records.
  • Raider page is designed to record daily raids. Use the Refresh button after the day has passed to reset the counters.
  • Collector page is designed to record you character collections. New release character's portrait image links will be provided in future posts.
  • Trader can be used to keep track of item prices and list of items you're planning to buy. If you are having trouble seeing the prices due to color, consider changing the color settings below.
  • Surveyor can be used to calculate family level.
  • Reminder is used as an alarm for important game events with time slots.
  • The page must remain open in order for the alarm to work.
  • Memoir can be used as a quick way to save texts whenever needed.
